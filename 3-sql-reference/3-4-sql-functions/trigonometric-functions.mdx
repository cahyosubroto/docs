| ------------- | ------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |

| `acosd`       | It calculates the inverse cosine of a given argument, where the output is expressed in degrees.  | `acosd(argument)`                                                             | > select acosd(0.5);It will return:>  acosd &#xA;-------&#xA;    60                                        |
