Aggregate functions compute a single result from a set of input values. Oxla supports the following functions:

| **Functions**                                        | **Description**                                                                                                                 |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- |
| [SUM](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/sum)          | Calculates and returns the sum of all values.                                                                                   |
| [MIN](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/min)           | Calculates and returns the minimum value.                                                                                       |
| [FOR\_MIN](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/sum)           | Calculates and returns a value corresponding to the minimal metric in the same row from a set of values.                        |
| [MAX](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/max)           | Calculates and returns the maximum value.                                                                                       |
| [FOR\_MAX](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/for-max)           | Calculates and Returns a value corresponding to the maximum metric in the same row from a set of values.                        |
| [AVG](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/avg)           | Calculates and returns the average value.                                                                                       |
| [COUNT](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/count)       | Counts the number of rows.                                                                                                      |
| [BOOL\_AND](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/bool-and) | Calculates the boolean of all the boolean values in the aggregated group. `FALSE` if at least one of aggregated rows is `FALSE` |
| [BOOL\_OR](https://oxla.mintlify.app/3-sql-reference/3-4-sql-functions/3-4-2-aggregation-functions/bool-or)   | Calculates the boolean of all the boolean values in the aggregated group. `TRUE` if at least one of aggregated rows is `TRUE`.  |

<Check>You can utilize the aggregate functions with the `GROUP BY` and `HAVING` clauses in the `SELECT` statement.&#x20;</Check>
