## Overview

The LOWER() function returns a given string, an expression, or values in a column in all lowercase letters. The syntax of the function is illustrated below:

```pgsql
LOWER(string)
```

It accepts input as a string and returns the text in the lowercase alphabet.

## Examples

### #Case 1: Basic `LOWER()` function

The following basic query converts the given string in all lowercase alphabets:

```pgsql
SELECT LOWER('PostGreSQL');
```

The final output will be as follows:

```pgsql
+------------+
| lower      |
+------------+
| postgresql |
+------------+
```

### #Case 2: `LOWER()` function using columns

Letâ€™s see how the `LOWER()` function works using an example with columns. We have a **personal\_details** table containing columns **id**, **first\_name**, **last\_name**, and **gender** of retail store employees.

```pgsql
CREATE TABLE personal_details (
  id int,
  first_name string,
  last_name string,
  gender string

    (4,'Emily','Byers','F'),
    (5,'Lucas','Sinclair','M');
```

```pgsql
SELECT * FROM personal_details;
```

The above query will show the following table:

```pgsql
+-----+-------------+-------------+----------+
| id  | first_name  | last_name   | gender   |
+-----+-------------+-------------+----------+
| 1   | Mark        | Wheeler     | M        |
| 2   | Tom         | Hanks       | M        |
| 3   | Jane        | Hopper      | F        |
| 4   | Emily       | Byers       | F        |
| 5   | Lucas       | Sinclair    | M        |
+-----+-------------+-------------+----------+
```

Letâ€™s assume that we want to convert the first and last names of employees with **id** numbers 2, 4, and 5 to all lowercase letters, which can be done using the following query:

